Movie Booking System

      This project is called a Movie Booking System. Our final project in our subject which is the Object Oriented Programming (OOP). The programming language used in this program is Python and Tkinter library is also used for the GUI of the program. This program is for booking or reserving movie tickets in the cinema. The program will ask about your information and the movie you would like to watch and after filling up the asked information, the admin can access the database in this program to see the records of the cinema and also its total sales.
