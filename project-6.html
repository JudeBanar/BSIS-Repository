Student Record System

This project is called Student Record System. This program or system is made using Python and Tkinter. I made this program to practice my skills and knwoledge about programming and Python language. In this program, you will be able to record your students' grades in a certain subjects and automatically get their average grade overall. This program also uses database to store all the data inputted in this system.
